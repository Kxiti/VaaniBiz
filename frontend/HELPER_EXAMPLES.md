## Integration Examples

### WhatsApp Share Flow

```
User clicks "Share on WhatsApp"
         ↓
Opens WhatsApp Web/App
         ↓
Message pre-filled in chat
         ↓
User selects contact
         ↓
Sends message
         ↓
Done! ✅
```

### Download Flow

```
User clicks "Download"
         ↓
Creates .txt file
         ↓
Filename: whatsapp-message-1234567890.txt
         ↓
Downloads to device
         ↓
User can open in any text editor
```

### Copy Flow

```
User clicks "Copy"
         ↓
Text copied to clipboard
         ↓
Button shows "Copied!" for 2 seconds
         ↓
User can paste anywhere
```
